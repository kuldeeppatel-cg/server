# E-commerce API

## Objective
Build a REST API using Express.js that manages E-commerce records stored in an in-memory JSON array.

## Implemented Routes

1. GET  /products
2. GET /states/:id
3. GET /states/highest-gdp
4. POST /states
5. PUT /states/:id
6. PUT /states/:id/budget
7. PUT /states/:id/population
8. PATCH /states/:id/literacy
9. PATCH /states/:id/gdp
10. PATCH /states/:id
11. DELETE /states/:id
12. DELETE /states/name/:stateName
13. DELETE /states/low-literacy/:percentage

## Sample URLs

http://localhost:3000/states <br/>
http://localhost:3000/states/:id <br/>
http://localhost:3000/states/highest-gdp  <br/>

## Steps to Run Locally

1. Clone repository
2. Run npm install
3. Run node server.js
4. Open browser or Postman at:
   http://localhost:3000
