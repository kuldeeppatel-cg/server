# E-commerce API

## Objective
Build a REST API using Express.js that manages E-commerce records stored in an in-memory JSON array.

## Implemented Routes

1. GET  /products
2. GET /priducts/:id
3. GET /products/category/:categoryName
4. POST /products
5. PUT /products/:id
6. PUT /products/:id/stock
7. PUT /products/:id/price

## Sample URLs

http://localhost:3000/products <br/>
http://localhost:3000/products/:id  <br/>
http://localhost:3000/products/category/:categoryName  <br/>
http://localhost:3000/products/:id/stock   <br/>
http://localhost:3000/products/:id/price        <br/>

## Steps to Run Locally

1. Clone repository
2. Run npm install
3. Run node server.js
4. Open browser or Postman at:
   http://localhost:3000
