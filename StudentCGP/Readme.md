# Student CGPA API

## Objective
Build a REST API using Express.js that manages student CGPA records stored in an in-memory JSON array.

## Implemented Routes

1. GET /students
2. GET /students/topper
3. GET /students/average
4. GET /students/count
5. GET /students/:id
6. GET /students/branch/:branchName
